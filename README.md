# Pet Shop Website

A responsive pet shop landing page built with plain HTML and CSS 
(no frameworks) as a self-study project — covering navbar, hero 
section, product cards, and a fully custom design system.

🔗 Live demo: https://github.com/Alan-projectworks/pet-shop-website-new

## What I practiced
- Flexbox and CSS Grid layouts
- A pure-CSS mobile navigation menu (no JavaScript)
- CSS animations (hover effects, floating hero image)
- Responsive design across screen sizes
